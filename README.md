# SAM THE QUESTIONER

A fun educational platformer game designed for primary school students to practice math while playing.

## 🎮 Game Description

This is a 2D platformer game where players run, jump, and solve math problems to progress through levels. 
The game combines classic platformer gameplay with educational math questions (addition, multiplication, division, and fractions).

Correct answers open portals to help you advance, while wrong answers send you back using lava checkpoints.

## ✨ Features

- Smooth player movement with running and jumping
- Animated character with walking animations
- Parallax scrolling backgrounds (3 layers)
- 5 levels with increasing difficulty
- Educational math questions integrated into the levels
- Portals for teleportation
- Lava hazards with checkpoints
- Animated trophy at the end
- Sound effects and background music
- Coyote time for better jumping feel

## 🎯 Learning Objectives

- Practice basic arithmetic (addition, multiplication, division, fractions)
- Improve problem-solving skills
- Develop timing and coordination
- Learn through play

## 🕹️ How to Play

1. Use **A/D** or **Arrow Keys** to move left and right
2. Hold **Left Shift** to run faster
3. Press **W** or **Up Arrow** to jump
4. Reach the question boards and choose the correct answer by going through the correct portal
5. Avoid lava! It will send you back to the last checkpoint
6. Reach the glowing trophy at the end to win!

## 📁 Project Structure

- `Assets/` - Contains all images, sounds, and fonts
- `Assets/Character/` - Player sprites
- `Assets/Map_assets/` - Backgrounds and UI elements
- `Assets/Platform/` - Platform and wall sprites
- `Assets/Portal/` - Portal animation frames
- `Assets/Sound_effects/` - Music and sound effects

## 🛠️ Technologies Used

- Python
- Pygame library

## 🎓 Educational Value

This game transforms traditional math practice into an interactive experience. Students must solve math problems to progress, making learning engaging and rewarding.
